## Hypothesis Resolution

- H-100: kept as a reference baseline only; not tradable.
- H-101: rejected because it over-filtered to zero OOS trades.
- H-102: inconclusive because the parent produced no OOS trades.
- H-103: deferred because 15m data coverage is incomplete.
- H-104: deferred because 5m data coverage is incomplete.
- H-105: deferred because it depends on deferred 5m trigger coverage.
